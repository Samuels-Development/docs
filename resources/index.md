# Resources

Welcome to the documentation for all Samuel's Development scripts. Select a resource below to get started.

## Scripts

### [Shops Pro](/resources/shops/)
A comprehensive shop system featuring ownership, employees, stock management, loyalty programs, coupons, sales, and a fully featured management UI. Supports regular shops, pawn shops, armories, illegal shops, and restricted shops.

### [Horde Mission](/resources/horde/)
A wave-based horde survival game mode with multiple maps, group-based gameplay, a leveling system, mystery box weapons, and full multiplayer support.

### [Advanced Crafting](/resources/crafting/)
A complete crafting system with static and placeable workbenches, recipes, blueprints, tech trees, tool durability, and a persistent crafting queue.

### [Dumpster Diving](/resources/dumpsters/)
Loot dumpsters and trash bins, encounter the Hobo King, befriend rat companions, send them on expeditions, and recycle scrap at recycler stations.

### [Beekeeping](/resources/beekeeping/)
An advanced beekeeping system with hive placement, honey production from multiple regions, infection and aggression mechanics, protective clothing, and a collaborator system.

### [Multijob & Boss Menu](/resources/multijob/)
An advanced multi-job management system allowing players to hold multiple jobs simultaneously, with duty zones, boss menus, applications, society accounting, bonuses, leaderboards, and weekly goals.

### [Corner Selling](/resources/selling/)
A drug dealing and corner selling system with zone-based selling, a 5-level reputation system, delivery missions, bulk sale encounters, robbery mechanics, milestones, and money laundering.

### [Oxy Run](/resources/oxyrun/)
A NoPixel-inspired drug delivery system with supplier pickups, buyer deliveries, a 3-level reputation system, money washing, 23 routes, and a robbery minigame mechanic.

### [Bobcat Security Heist](/resources/bobcat/)
A multi-stage Bobcat Security weapons depot heist with thermite breaching, keycard doors, C4 vault bombing, weapon crate looting, and NPC guards. Supports Gabz, NoPixel, and K4MB1 MLOs.

### [Pacific Bank Heist](/resources/pacificbank/)
A comprehensive Pacific Bank heist featuring computer hacking, door consoles, laser grids, thermite breaching, vault bombing, cash tray looting, and deposit box drilling.

### [Yacht Heist](/resources/yacht/)
A puzzle-based yacht heist with security code entry, pressure management, screen sequencing, laptop hacking, password puzzles, vault access, and a cinematic beach washup ending.

### [Coke Mission](/resources/cokemission/)
A NoPixel-inspired cocaine trafficking mission with underwater vehicle drops, sealed cache looting, an optional phone booth minigame, and flare effects.

### [Traphouse Robbery](/resources/traphouse/)
A Vagos gang traphouse robbery with 22 lootable props, vault cracking, NPC guards, and hacking minigames for cash, drugs, weapons, and valuables.

### [Warehouse Heist](/resources/warehouse/)
A secured warehouse heist with thermite breaching, interior exploration, laptop hacking, six loot crates, optional NPC guards, and a two-exit escape system.

## Common Dependencies

All scripts require one of the following frameworks:

| Dependency | Options |
|---|---|
| **Framework** | `qb-core` / `qbx_core` / `es_extended` |
| **Target System** | `ox_target` / `qb-target` / `qtarget` |
| **Library** | `ox_lib` |
| **Database** | `oxmysql` |

::: tip
Most scripts auto-detect your framework and handle database setup automatically on first start.
:::
